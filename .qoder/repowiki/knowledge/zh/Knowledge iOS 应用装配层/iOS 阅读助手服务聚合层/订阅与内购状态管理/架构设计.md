单一文件单类实现：`SubscriptionManager` 作为 `@MainActor final class` 暴露全局单例 `shared`，通过 `ObservableObject` + `@Published` 属性（`isPremium`、`isLoading`、`products`、`currentSubscription`）向 SwiftUI 视图层推送状态变化。

- 依赖方向单向：仅依赖 Foundation 与 StoreKit，不反向引用业务层；产品 ID 以常量数组集中配置在 `productIDs`，便于替换为 App Store Connect 中的真实 ID。
- 生命周期：私有 init 中启动两个并发 Task 分别调用 `loadProducts()` 与 `checkSubscriptionStatus()`，完成初始化即进入就绪态。
- 对外 API 边界清晰：`loadProducts` / `purchase(_:)` / `restorePurchases()` 三个公开方法，内部验证逻辑封装在私有 `checkVerified(_:)`，错误收敛到本地 `StoreError.failedVerification`。
- 状态一致性：购买成功后显式调用 `transaction.finish()` 并刷新 `checkSubscriptionStatus()`，确保 UI 与 App Store 状态同步。
