- 每个对外服务类都声明 `static let shared = XxxService()` 私有 init() 的单例，并通过该单例被上层消费。
- 网络型 Service（AISummaryService、CompanionService、CosyVoiceService）内部统一用 `UserDefaults.standard.string(forKey: "dashscope_api_key")` 读取密钥，并以 `AIServiceError`/`CosyVoiceError` 等 LocalizedError 枚举返回可本地化的错误描述。
- HTTP 请求构建遵循相同模板：构造 URLRequest → 设置 Content-Type/Bearer Authorization → JSONSerialization 编码 body → await session.data(for:) → 校验 HTTPURLResponse 状态码 → 解析 output.choices[0].message.content。
- 异步回调一律使用闭包属性（如 onPositionChange、onRangeChange、onError），并在 DispatchQueue.main.async 中派发以保证 UI 线程安全。
- 长文本处理采用「先截断再自然断点」策略：AISummaryService 取前 8000 字符，SpeechService 每 500 字符一段并在句号/换行处对齐。
