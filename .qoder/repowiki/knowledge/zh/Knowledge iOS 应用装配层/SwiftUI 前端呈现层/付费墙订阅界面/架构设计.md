单一 SwiftUI View（PaywallView）作为入口，通过 @ObservedObject 持有全局 SubscriptionManager.shared 读取产品列表、加载状态与 isPremium 标志，自身仅负责 UI 渲染与交互转发：
- body 以 NavigationStack + ScrollView 组织顶部图标、标题、featureRow 列表、subscriptionOptions 以及恢复购买按钮；
- subscriptionOptions 使用 @ViewBuilder 根据 isLoading / products.isEmpty / 正常三种分支渲染 ProgressView、重试提示或 ForEach 产品卡片；
- purchaseProduct 与恢复购买均封装为 Task 异步调用 SubscriptionManager.purchase / restorePurchases，成功则 dismiss() 关闭弹窗。
依赖方向单向：View → SubscriptionManager → StoreKit，View 不直接 import 除 SwiftUI/StoreKit 外的业务层。
