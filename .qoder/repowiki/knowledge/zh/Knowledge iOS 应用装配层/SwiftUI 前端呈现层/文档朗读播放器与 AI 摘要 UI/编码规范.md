- 所有 UI 状态数据统一通过 `@ObservedObject var speakerVM: SpeakerViewModel` 注入，View 自身不持有播放状态，仅触发 VM 方法。
- 可复用的交互元素封装为私有嵌套 View（如 `ControlButton`），通过枚举参数（Size.small/large）区分尺寸，避免在父 View 中重复写 Button 样式。
- 异步/多态 UI 采用 `@ViewBuilder` + if-else 分支渲染（如 `summarySheet` 根据 isGeneratingSummary / summaryError / summaryResult 切换 Loading/Error/Card 三种状态）。
- UI 文案与系统图标统一使用 SF Symbols（Image(systemName:)）配合 Label 组件，保持图标语义一致。
