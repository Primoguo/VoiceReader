本模块为纯数据层，按业务域拆分为独立 Swift 文件，仅依赖 Foundation / SwiftData / SwiftUI，被上层 UI 与逻辑层单向消费。
- 文档域：`Document.swift` 使用 `@Model` 标注的 SwiftData 实体，通过 `fileTypeRaw` 字符串字段 + `fileType` 计算属性桥接 `DocumentType` 枚举；`SummaryResult.swift` 提供 `toJSON()` / `fromJSON(_:)` 静态工厂用于摘要 JSON 编解码。
- 语音域：`ClonedVoice.swift` 同时承载 `PresetVoice`（CosyVoice 内置）与 `ClonedVoice`（用户克隆）两个 Codable 结构体，并内嵌 `VoiceStore` 单例式管理器，基于 `UserDefaults` 保存克隆列表与选中 ID，并提供 `presetVoices` 常量清单与 `presetsByCategory()` 分组查询；`VoiceConfig.swift` 定义 `TTSEngine` 引擎枚举（system/edgeTTS/knowledgeVoice/legacySystem）及合成参数结构体。
- 应用设置域：`ThemeMode.swift` 将主题映射到 SwiftUI `ColorScheme`；`PlaybackState.swift` 是极简的状态枚举。
- 伴读对话：`CompanionMessage.swift` 提供无状态的聊天消息结构体。
- 依赖方向严格单向向下：本模块不引用任何上层模块，仅依赖系统框架。
