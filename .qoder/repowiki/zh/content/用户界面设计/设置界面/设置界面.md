## 简介
本文件面向开发者与维护者，系统化梳理"设置"相关界面的整体布局、功能组织与数据流，覆盖以下要点：
- SettingsView的多引擎动态适配与分组交互逻辑，包括即将推出功能的特殊处理和Premium订阅管理
- AI伴读难度级别选择功能，支持入门、进阶、专业三种回答风格的个性化配置
- Edge TTS引擎集成，提供免费的微软Neural云端语音合成服务
- SubscriptionManager的订阅状态管理与StoreKit 2集成
- PaywallView的付费墙界面设计和购买流程
- VoiceSelectView的AI音色选择与参数调节联动
- SystemVoiceSelectView的Apple Neural TTS专业音色管理与下载引导
- EdgeVoiceSelectView的云端音色选择和试听功能
- VoiceCloneView的声音克隆流程与用户引导
- LycheeLevelManager的荔枝成长体系管理
- 主题切换、语言设置与其他配置的持久化策略
- 设置验证、默认值恢复与配置导入导出能力（含实现建议）
- 新增设置项与自定义配置界面的开发规范
- **新增** 隐私政策页面集成和用户隐私保护措施说明

## 项目结构
设置相关的代码主要分布在 Views、Models、Services 与 ViewModels 四个层次：
- Views：提供 UI 与用户交互（设置主界面、付费墙、音色选择、语音克隆、系统音色管理、AI伴读难度选择、Edge云端音色管理）
- Models：定义配置模型、主题枚举、预设与克隆音色数据结构、荔枝等级管理器
- Services：主题管理、CosyVoice 服务、系统语音管理、错误处理、订阅管理、AI伴读服务、Edge TTS服务和合成器
- ViewModels：统一编排播放引擎、配置加载与保存、状态同步

## 核心组件
- **SettingsView**：设置主入口，支持多TTS引擎动态适配，按引擎类型显示不同的语音选择界面；实时应用并持久化配置；**已升级** Premium订阅状态显示和付费墙访问入口，采用轻量级图标和次级颜色背景的简洁设计；AI伴读难度级别选择功能，使用分段控件实现入门、进阶、专业三种回答风格的切换；Edge TTS引擎支持，显示绿色"免费"徽章；即将推出功能的灰化处理；移除 API密钥配置功能；**新增** 隐私政策链接集成
- **CompanionService**：AI伴读服务，管理对话历史、学习状态和难度级别配置，通过服务器中转调用通义千问，支持多轮对话和上下文感知
- **LycheeLevelManager**：荔枝成长体系管理器，根据累计收听时长计算荔枝等级，提供实时追踪和持久化存储
- **EdgeVoiceSelectView**：专为Edge TTS设计的云端音色选择界面，展示精选中文和粤语音色列表，支持试听功能和性别标识
- **EdgeTTSService**：Edge TTS云端语音合成服务，通过自建服务器中转调用微软Edge TTS，提供免费无限量的高质量语音合成
- **EdgeTTSSynthesizer**：Edge TTS引擎适配器，实现SpeechSynthesizerProtocol，支持分段合成、位置追踪和错误降级
- **SubscriptionManager**：订阅管理器，基于StoreKit 2实现订阅状态检查、产品加载、购买和恢复功能
- **PaywallView**：付费墙界面，展示Premium功能列表和订阅选项，提供完整的购买流程
- **SystemVoiceSelectView**：专为iOS 17+ Apple Neural TTS设计的音色选择界面，提供推荐音色分类和专业音质标识；增强下载引导和用户提示功能
- VoiceSelectView：展示"我的音色"和"预设音色"，支持试听、删除、选择并切换引擎
- VoiceCloneView：录音→预览→上传→AI 学习→完成的全流程引导，包含权限校验、时长校验与错误提示
- ThemeManager：主题模式单例，读写 UserDefaults，驱动全局主题切换
- SpeakerViewModel：统一暴露 TTS 控制接口，负责引擎切换、配置加载/保存、播放状态同步与降级策略；增加Edge TTS引擎支持和错误降级逻辑
- CosyVoiceService：封装云端 TTS 与克隆接口，返回音频或 voice_id，集中错误类型
- SystemVoiceManager：系统语音管理器，提供iOS 17+ Neural TTS音色查询和推荐算法
- ErrorHandler：统一错误日志与弹窗提示

## 架构总览
设置界面采用 MVVM + Service 分层，支持多引擎动态适配、AI伴读难度管理和订阅管理：
- 视图层通过 @ObservedObject/@State 绑定 ViewModel 与服务，响应式更新 UI
- 配置以 JSON 编码后写入 UserDefaults，保证跨会话持久化
- AI伴读难度级别通过 CompanionService.difficulty 属性管理，自动持久化到 UserDefaults
- 引擎切换由 SpeakerViewModel 统一调度，必要时自动降级到系统 TTS
- 云端能力通过 CosyVoiceService 和 EdgeTTSService 调用，错误集中抛出并由 ErrorHandler 呈现
- 订阅管理通过 SubscriptionManager 统一管理，使用 StoreKit 2 进行 IAP 处理
- 设备能力检查通过 TTSEngine.isSupported 属性在渲染前过滤不支持的引擎选项
- 即将推出功能通过专门的视图构建器处理禁用状态
- 荔枝成长体系通过 LycheeLevelManager 管理累计收听时长和等级变化

## 详细组件分析

### SettingsView：多引擎动态适配的设置主界面
- **更新** 分组与布局
  - 外观：遍历 ThemeMode 列表，点击即切换主题，右侧显示选中标记
  - **已升级** Premium 功能：采用轻量级图标和次级颜色背景的简洁设计，未订阅用户显示带有浅色背景图标的卡片，包含副标题描述"AI 总结、AI 伴读、高品质音色"，提供更好的视觉层次结构；已订阅用户显示皇冠图标和"Premium 已激活"状态，附带"全部 AI 功能已解锁"的详细说明
  - AI伴读设置：独立的"AI 伴读"分组，包含回答风格选择器和详细说明
    - 使用分段控件实现入门、进阶、专业三种难度级别的直观选择
    - 每个级别配有详细的风格说明，帮助用户理解不同选项的特点
    - 实时更新 CompanionService.shared.difficulty 并持久化存储
  - 语音引擎：使用 `TTSEngine.allCases.filter { $0.isSupported }` 动态过滤支持的引擎；每个引擎显示名称和描述；Knowledge Voice 现在显示"即将推出"标签和锁图标，处于禁用状态；Apple Neural TTS 标记为"推荐"选项；Edge TTS 显示"免费"绿色徽章
  - 智能语音选择：根据当前选择的引擎类型显示对应的语音选择界面
    - Apple Neural TTS：显示"系统音色"按钮，打开 SystemVoiceSelectView
    - Knowledge Voice：显示"AI 音色"按钮，打开 VoiceSelectView  
    - Edge TTS：显示"云端音色"按钮，打开 EdgeVoiceSelectView，显示当前选中的云语音名称
    - 传统系统 TTS：显示传统的语言和声音 Picker
  - 朗读设置：语速滑块+快捷档位、音高滑块、音量滑块；变更时即时 apply()
  - 关于：显示版本号；**新增** 隐私政策链接，点击可访问完整的隐私政策页面
- **更新** 数据流
  - onAppear 从 speakerVM.voiceConfig 初始化本地 State，同时从 CompanionService.shared.difficulty 加载AI伴读难度设置
  - apply() 构建 VoiceConfig 并写入 UserDefaults，若处于播放中则调用 updateConfig 热更新
  - saveConfig() 将完整配置 JSON 编码保存
- **新增** 关键点
  - 使用 AVSpeechSynthesisVoice.speechVoices() 动态获取可用语音并按语言前缀过滤
  - 当引擎为 Knowledge Voice 时，VoiceConfig 会附带 selectedCloneId/presetVoiceId
  - 设备能力检查：通过 `TTSEngine.isSupported` 属性在 iOS < 17 时自动过滤 Apple Neural TTS
  - 条件渲染：根据 selectedEngine 的值动态显示相应的语音选择入口
  - comingSoonEngineRow() 视图构建器：为即将推出的功能提供统一的禁用状态处理
  - 订阅状态监听：通过 @ObservedObject subscriptionManager 实时响应订阅状态变化
  - Edge TTS徽章显示：当选中Edge TTS引擎时，显示绿色"免费"标签
  - AI伴读难度级别管理：通过分段控件实现直观的三级难度选择，实时更新并持久化
  - **新增** 隐私政策集成：在"关于"部分添加隐私政策链接，指向 naolizhi.cn/privacy.html
- **移除** API密钥配置：不再在设置界面中显示API Key输入和保存功能

## 依赖关系分析
- 视图对 ViewModel 的依赖

## 性能与体验优化
- 避免频繁重绘
  - 语速/音高/音量滑动时使用 .onChange 触发最小必要更新，已在 SettingsView 中实现

- 引擎切换

## 故障排查指南
- 录音失败
  - 现象：需要麦克风权限或录音时长不足
  - 处理：授予麦克风权限；确保录音≥5秒
- 引擎异常降级
  - 现象：Knowledge Voice 或 Edge TTS 出错后自动回退系统 TTS
  - 处理：查看错误日志；确认网络与 API Key；必要时手动切回系统 TTS
- Edge TTS相关问题
  - 现象：Edge TTS合成失败或音色无法试听
  - 处理：检查网络连接；确认Edge TTS服务器可用性；查看控制台错误日志
  - 现象：Edge TTS音色选择界面空白
  - 处理：重启应用；检查EdgeTTSService的音色列表初始化
- AI伴读相关问题
  - 现象：AI伴读难度设置不生效
  - 处理：检查CompanionService.difficulty属性是否正确更新；确认UserDefaults存储正常
  - 现象：AI回答风格不符合预期
  - 处理：检查难度级别设置；确认系统提示词中的难度自适应逻辑
  - 现象：对话历史丢失
  - 处理：检查conversationHistory数组长度限制；确认resetConversation()调用时机
- 荔枝成长体系问题
  - 现象：累计收听时长不更新
  - 处理：检查LycheeLevelManager.startTracking()是否正确调用；确认Timer是否正常运行
  - 现象：等级不提升
  - 处理：检查累计时长是否达到等级阈值；确认等级计算逻辑
- 订阅相关问题
  - 现象：付费墙无法加载或购买失败
  - 处理：检查网络连接；确认App Store Connect配置；查看控制台错误日志
  - 现象：订阅状态不正确
  - 处理：重启应用重新检查订阅状态；使用"恢复购买"功能；检查设备时间设置
- 设备兼容性问题
  - 现象：iOS < 17 设备上无法选择 Apple Neural TTS 或看到升级提示
  - 处理：这是预期行为，系统会自动过滤不支持的引擎选项；建议升级到 iOS 17+
- 系统音色选择问题
  - 现象：SystemVoiceSelectView 显示空白或无法加载音色
  - 处理：检查 iOS 版本是否为 17+；确认网络连接正常；重启应用重试
- Neural TTS 音色下载问题
  - 现象：SystemVoiceSelectView 显示下载提示但无法找到Neural音色
  - 处理：按照提示前往系统设置手动下载所需的中英文Neural TTS音色包
- Premium订阅卡片显示问题
  - 现象：订阅卡片显示异常或状态不正确
  - 处理：检查SubscriptionManager状态；确认订阅状态是否正确同步；重启应用刷新状态
- **新增** 隐私政策链接问题
  - 现象：点击隐私政策链接无反应或无法打开
  - 处理：检查网络连接；确认naolizhi.cn域名可访问；检查URL格式是否正确
  - 现象：隐私政策页面显示异常
  - 处理：检查网页内容完整性；确认HTML格式正确；检查CSS样式是否正常加载
- 配置丢失
  - 现象：重启后设置未生效
  - 处理：检查 UserDefaults 键值是否存在；确认 JSON 编解码是否成功
- AI伴读配置丢失
  - 现象：重启后AI伴读难度设置重置
  - 处理：检查companion_difficulty键值；确认CompanionService.difficulty属性的get/set逻辑
