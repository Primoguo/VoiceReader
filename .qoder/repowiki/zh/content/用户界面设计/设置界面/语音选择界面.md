## 简介
本文件围绕"语音选择界面"展开，聚焦 VoiceSelectView 的展示与交互、EdgeVoiceSelectView 的云端音色管理、SystemVoiceSelectView 的系统 TTS 管理、多引擎架构的统一编排、语言筛选与预览机制、实时参数调节接口、配置持久化与同步、测试播放流程以及性能优化策略。同时提供自定义语音引擎扩展指南，帮助开发者在现有架构上平滑接入新的合成后端。

## 项目结构
与语音选择相关的代码主要分布在以下模块：
- Views：UI 层，包含音色选择页面 VoiceSelectView、云端音色选择页面 EdgeVoiceSelectView 和系统音色选择页面 SystemVoiceSelectView
- Models：数据模型与存储，包括 VoiceConfig、PresetVoice、ClonedVoice、VoiceStore
- ViewModels：业务编排与状态管理，SpeakerViewModel 作为门面协调引擎与 UI
- Services：引擎实现与网络服务，SpeechService（系统 TTS）、CosyVoiceService（阿里云 CosyVoice HTTP API）、CosyVoiceSynthesizer（引擎适配器）、EdgeTTSService（Edge TTS 云端服务）、EdgeTTSSynthesizer（Edge TTS 适配器）、SystemVoiceManager（系统语音管理器）
- 辅助服务：LanguageDetector 自动检测文档语言并匹配语音

## 核心组件
- VoiceSelectView：负责展示预设音色与克隆音色列表、用户选择交互、试听播放、删除克隆音色等。
- **新增** EdgeVoiceSelectView：专门管理 Edge TTS 云端音色选择，提供普通话和粤语分类展示、性别标识、标签系统和试听功能。
- SystemVoiceSelectView：专门管理 iOS 17+ Apple Neural TTS 音色选择，提供智能推荐排序和下载引导。
- SpeakerViewModel：统一编排播放控制、引擎切换、配置加载与保存、错误降级、远程控制同步。
- SpeechSynthesizerProtocol：定义引擎抽象接口，屏蔽系统 TTS 与 AI 引擎差异。
- SpeechService：基于 AVSpeechSynthesizer 的系统 TTS 实现。
- CosyVoiceSynthesizer：将 CosyVoiceService 封装为引擎协议，支持分段合成与流式播放。
- **新增** EdgeTTSSynthesizer：将 EdgeTTSService 封装为引擎协议，支持分段合成与流式播放。
- **新增** EdgeTTSService：Edge TTS 云端服务，提供精选中文和粤语音色列表及合成能力。
- CosyVoiceService：HTTP 客户端，调用阿里云 DashScope CosyVoice 接口进行合成与克隆。
- SystemVoiceManager：系统语音管理器，提供 Neural TTS 音色过滤、推荐和检测功能。
- LanguageDetector：根据文本主导语言自动匹配系统语音或保持当前配置。
- VoiceConfig/PresetVoice/ClonedVoice/VoiceStore：语音配置、预设与克隆数据模型及本地持久化。

### VoiceSelectView：AI 音色选择与交互
- 列表组织
  - "我的音色"：展示已克隆的音色，支持删除操作。
  - "录制我的声音"：打开 VoiceCloneView 进行录音与克隆。

  - 选择预设音色：设置 engine=knowledgeVoice、presetVoiceId、清空 clonedVoiceId，并持久化选择。

- 初始化与恢复
  - onAppear 中加载当前选择与克隆列表，确保 UI 与持久化一致。

### EdgeVoiceSelectView：Edge TTS 云端音色管理
**新增** 专门处理 Edge TTS 云端音色的选择与管理

- 分类展示
  - 普通话音色：展示 16 种精选中文女声和男声，按性别分组排列
  - 粤语音色：展示 3 种粤语方言音色，带有"粤语"标签标识
- 视觉设计
  - 性别标识：女性音色显示粉色人物图标，男性音色显示蓝色人物图标
  - 标签系统：支持"推荐"、"新闻"、"粤语"等彩色标签，提升辨识度
  - 选中标记：当前选中的音色显示对勾标记，未选中显示空心圆圈
- 试听功能
  - 每行右侧都有试听按钮，点击后调用 EdgeTTSService.synthesize 生成示例音频
  - 试听文本自动生成："你好，这是[音色名称]的声音效果。"
  - 试听期间禁用其他操作，防止并发冲突
- 选择与应用
  - 点击音色行即可选中，设置 engine=edgeTTS 和对应的 edgeVoiceId
  - 点击确定按钮应用选择并关闭页面
  - 支持取消操作返回上一页
- 默认值处理
  - 首次使用时默认选择晓晓（zh-CN-XiaoxiaoNeural），最自然的中文女声
  - 从持久化配置中恢复上次选择的音色

### SystemVoiceSelectView：Apple Neural TTS 智能管理
**新增** 专门处理 iOS 17+ Apple Neural TTS 的高级功能

### 实时参数调节接口：语速、音高、音量即时反馈

- 系统 TTS 与 AI 引擎的差异

  - **新增** Edge TTS：rate 转换为百分比格式（+0% ~ +100%）传入服务端；pitch/volume 由播放器控制。

## 性能考虑
- 分段合成与播放
  - CosyVoiceSynthesizer 将长文本切分为不超过 500 字符的段落，逐段合成并拼接播放，降低单次请求负载与内存占用。
  - **新增** EdgeTTSSynthesizer 将长文本切分为不超过 2000 字符的段落（Edge TTS 服务器限制 5000），逐段合成并拼接播放。
- 自然断点优化
  - 分段算法尝试在句号、换行、逗号、空格等自然断点处截断，提升朗读流畅度。
- 网络节流
  - 多段合成间加入短延迟，避免请求过快导致服务端限流。
- 资源清理
  - 预览与合成音频均写入临时目录，播放结束后及时释放播放器与任务，避免泄漏。
- 估算位置
  - 对于非系统 TTS 引擎，采用每秒约 3 个字符的粗略估算来更新位置，减少频繁计算开销。
- **更新** 智能缓存与预加载
  - SystemVoiceManager 缓存系统语音列表，避免重复查询开销。
  - 推荐的 Neural 音色预加载，提升首次响应速度。
  - **新增** Edge TTS 音色列表静态缓存，避免重复构建开销。

## 结论
VoiceSelectView 提供了直观的 AI 音色选择与试听能力，**新增的** EdgeVoiceSelectView 专门管理 Edge TTS 云端音色，SystemVoiceSelectView 专门管理 Apple Neural TTS 的高级功能，结合 SpeakerViewModel 的统一编排与 SpeechSynthesizerProtocol 的抽象设计，实现了系统 TTS、Edge TTS 与 AI 引擎的无缝切换。**更新的** 四引擎架构（系统 TTS、Edge TTS、Knowledge Voice、传统系统 TTS）为用户提供了更多选择，其中 Edge TTS 作为免费云端方案，提供了丰富的中文和粤语音色。智能推荐排序优先展示中文 Neural 音色，增强了用户体验。语言自动检测与配置持久化确保了跨设备与跨会话的一致性。通过分段合成、网络节流、资源清理和智能缓存等策略，系统在性能与稳定性方面具备良好表现。未来可通过扩展自定义引擎进一步提升生态兼容性。
